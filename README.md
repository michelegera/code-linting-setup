## Code linting tools

This a template to set up standard code linting tools for JS projects in my
preferred configuration.

```bash
# Default
yarn add -D \
  eslint \
  eslint-config-prettier \
  eslint-plugin-import \
  eslint-plugin-prettier \
  husky \
  lint-staged \
  prettier

# Add these for React/Next.js
yarn add -D \
  eslint-plugin-react
```
